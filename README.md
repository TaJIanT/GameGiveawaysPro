# 🎮 GameGiveawaysPro

**GameGiveawaysPro** — бесплатное десктопное приложение для отслеживания бесплатных игр и выгодных скидок на популярных игровых платформах.

![Version](https://img.shields.io/badge/version-1.0.3-blue)
![Python](https://img.shields.io/badge/python-3.10%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 📋 Возможности

- 🎁 **Отслеживание бесплатных раздач** игр на Steam, Epic Games, GOG
- 💰 **Мониторинг скидок** до 95% на популярных платформах
- 🔄 **Автоматическое обновление** списка игр
- 📊 **Удобная фильтрация** по платформам (отдельные вкладки)
- 🌐 **Прямые ссылки** на страницы игр для быстрого получения
- 🎨 **Современный темный интерфейс** с неоновыми акцентами
- 💾 **Кеширование данных** для работы без интернета
- 🔔 **Система автообновлений** приложения (GitHub Releases)

## 📥 Установка

### Вариант 1: Готовый EXE-файл (Windows)

1. Перейдите в раздел [Releases](https://github.com/TaJIanT/GameGiveawaysPro/releases)
2. Скачайте последнюю версию `GameGiveawaysPro.exe`
3. Запустите файл — установка не требуется

### Вариант 2: Запуск из исходников

1. Клонируйте репозиторий:
```bash
git clone https://github.com/TaJIanT/GameGiveawaysPro.git
cd GameGiveawaysPro
Установите зависимости:

bash
pip install -r requirements.txt
Запустите приложение:

bash
python main.py
🛠️ Системные требования
ОС: Windows 10/11 (для EXE) или любая ОС с Python

Python: 3.10 или выше (для запуска из исходников)

Интернет: требуется для обновления списка игр и проверки обновлений

RAM: минимум 256 МБ

Разрешение экрана: рекомендуется 1920x1080 и выше

🎯 Использование
Запустите приложение — список игр загрузится автоматически

Выберите вкладку: Все / Steam / Epic / GOG / Скидки

Кликните на карточку игры для просмотра подробной информации

Нажмите "Открыть в браузере" для перехода на страницу игры

Используйте кнопку "Обновить" для обновления списка

Настройки
GamerPower — включите/отключите источник данных GamerPower API для раздач Steam

🔧 Технологии
GUI: CustomTkinter — современный UI-фреймворк

API Sources:

CheapShark API — скидки и предложения

GamerPower API — бесплатные раздачи

Автообновления: GitHub Releases API

Сборка: PyInstaller

📂 Структура проекта
text
GameGiveawaysPro/
├── main.py              # Главный файл приложения
├── api.py               # Логика работы с API (CheapShark, GamerPower)
├── cards.py             # Компоненты карточек игр
├── tabs.py              # Система вкладок

├── config.py            # Конфигурация и темы
├── storage.py           # Работа с локальным хранилищем

├── updater.py           # Модуль обновления
├── build.ps1            # Скрипт сборки
├── requirements.txt     # Зависимости Python
└── README.md            # Документация
